# US-096: Track Individual Skill Assessments

**As an** Engineering Team Lead,
**I want to** track individual team members' skill levels for each practice area,
**so that** I can identify skill gaps and create targeted development plans.

## Context

Individual adoptions record a person's competency level for a given practice area. Each record captures the person name, practice area, competency level, supporting evidence, and free-form notes. The by-person query endpoint enables viewing all practice area competency levels for a single individual, providing a personal development dashboard view.

## Road Items

- ROAD-048 (Practice Area, Competency & Adoption CRUD)

## Acceptance Criteria

1. GET `/api/v1/taxonomy/individual-adoptions` returns all individual adoption records across practice areas
2. GET `/api/v1/taxonomy/individual-adoptions/by-person/:personName` returns all adoption records for a specific person
3. POST `/api/v1/taxonomy/individual-adoptions` creates a new individual adoption record with competency level, evidence, and notes
4. PUT `/api/v1/taxonomy/individual-adoptions/:id` updates an existing individual adoption record to reflect skill progression
5. DELETE `/api/v1/taxonomy/individual-adoptions/:id` removes an individual adoption record from the system

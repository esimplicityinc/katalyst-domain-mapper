# US-090: View Layer Health Dashboard

**As a** Non-Technical Stakeholder,
**I want to** view a dashboard showing the health of each system layer with simple pass/warn/fail indicators,
**so that** I can understand where quality risks exist without reading technical metrics.

## Context

This story provides the consumer-facing view for the Layer Health capability (CAP-031). While Engineering Team Leads use US-089's detailed metrics for technical decision-making, Non-Technical Stakeholders need a simplified dashboard with color-coded health indicators and expandable detail. The dashboard maps directly to the `LayerHealthSchema` structure: each card represents a taxonomy layer, the overall color reflects the worst-performing category, and drill-down reveals the three dimension scores with links to governing NFRs.

## Acceptance Criteria

1. The dashboard displays each taxonomy layer as a card with its `overallStatus` shown as a color-coded indicator (green/amber/red)
2. Cards show the `overallScore` as a percentage
3. Clicking a card expands to show the three category scores (understandability, functionality, compliance)
4. Category scores link to their governing NFRs for context on what "pass" means
5. The dashboard loads within 2 seconds for up to 50 layers

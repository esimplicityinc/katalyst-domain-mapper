# US-073: Switch Between Layout Engines

**As an** Engineering Team Lead,
**I want to** switch between three layout algorithms for the landscape canvas,
**so that** I can choose the layout that best represents my domain's structure and relationships.

## Context

Complex domain graphs have varying topologies — some are highly hierarchical (benefiting from ELK's layered layout), others have flat relationships (Dagre works well), and some have organic cluster structures (D3-Force reveals natural groupings). The multi-engine approach (ADR-018) allows users to pick the right visualization for their mental model of the domain.

## Road Items

- [ROAD-044](../roads/ROAD-044) — Business Landscape Graph

## Acceptance Criteria

1. The landscape toolbar displays three layout engine buttons: ELK (default), Dagre, and D3-Force
2. Clicking a layout engine button re-runs the layout and re-renders the canvas with the new engine's positions
3. The toolbar displays the layout engine's execution time in milliseconds after each layout computation
4. ELK layout groups bounded contexts under their taxonomy system hierarchy boxes
5. Dagre layout produces a top-down directed acyclic graph ordering
6. D3-Force layout produces an organic force-directed layout

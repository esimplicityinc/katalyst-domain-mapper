# User Types

User types represent the key actors who interact with the Katalyst Delivery Framework. Each user type has defined goals, pain points, and capabilities they rely on.

## User Type Types

| Type | Description |
|------|-------------|
| `human` | A person who uses the system directly |
| `bot` | An AI agent or automated process |
| `system` | An infrastructure component |
| `external_api` | An external service that exchanges data with the system |

# CAP-027: Reusable AI Chat Component Library

## Description

The `@katalyst/chat` package provides a shared, configurable chat component library that eliminates duplicated AI chat code across the application. It exports `OpenCodeChat` (a styled React component) and `useOpenCodeChat` (a headless hook), enabling consistent AI chat experiences on the FOE assessment, DDD modeling, and Taxonomy management pages. The component handles SSE streaming, markdown rendering with syntax highlighting, session lifecycle, and the question/answer tool pattern ‚Äî all configurable via props like `agentId` and `accentColor`.

## Key Features

- **`OpenCodeChat` component**: Drop-in chat UI with markdown rendering, SSE streaming, suggestion chips, dark mode, and accent color variants (`purple`, `blue`, `green`, `rose`)
- **`useOpenCodeChat` hook**: Headless state management for session creation, message sending, SSE event handling, and question/answer tool support
- **`createClient` API helper**: Low-level wrapper around the OpenCode SDK for health checks and session operations
- **Agent targeting**: `agentId` prop routes conversations to different AI agents (`foe-assessment-coach`, `ddd-domain-mapper`, `taxonomy-architect`)
- **3 consumers**: FOEChat, DDDChat, and TaxonomyChat all use `@katalyst/chat` with zero duplicated logic

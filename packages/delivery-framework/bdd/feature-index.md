# Feature Index

All BDD feature files organized by bounded context.

## Organization

Feature files are stored in `stack-tests/features/` and organized by context:

```
stack-tests/
  features/
    api/          # API integration tests
    ui/           # UI acceptance tests
    hybrid/       # Full-stack tests
```

## Feature Files

Feature files will be listed here as they are created.
